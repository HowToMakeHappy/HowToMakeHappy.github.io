## 프로젝트 만들기

```
vue create 프로젝트명
vue create todo
```

## Manually 설정

```
Manually select features

Babel
Router
Vuex
Linter / Formatter

History mode : yes
ESLint + Standard config
config file : dedicated config fils
use yarn

cd todo
yarn serve
http://localhost:8080/
```
