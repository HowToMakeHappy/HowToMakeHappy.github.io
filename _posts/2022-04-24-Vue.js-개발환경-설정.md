# node.js 설치
구글에서 node.js 검색 후 설치 진행

<br />

## yarn 설치
```
npm i -g yarn
```

<br />

## git 설치
```
구글에서 git 검색 후 설치 진행
```
<br />

## 설치 확인
```
node -v 
npm -v
git --version 
```
 
<br />
<br />

# vue cli 설치
```
npm install -g @vue/cli
# OR
yarn global add @vue/cli
```

<br />

# vue cli 설치
```
vue --version
```

<br />

## 확인 결과

```
~ % vue --version
@vue/cli 5.0.4
```
